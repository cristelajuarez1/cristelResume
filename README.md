# 💼 Cristela Juárez — Resume Website

Sitio web personal que presenta mi perfil profesional como **Analista de Datos y Desarrolladora**, incluyendo experiencia, habilidades y proyectos relevantes.

## 🌐 Demo

Puedes ver el sitio en línea aquí:
👉 https://cristelajuarez1.github.io/cristelResume/

## 🧠 Sobre el proyecto

Este proyecto es un sitio web estático construido a partir del template **Strata** de HTML5UP, personalizado para reflejar mi identidad profesional, experiencia y enfoque en análisis de datos y desarrollo.

El objetivo es contar con una presencia digital clara, accesible y profesional.

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Template: HTML5UP (Strata)

```bash
.

```

---

## 🚀 Cómo ejecutar el proyecto

1. Clona el repositorio:

```bash
git clone https://github.com/cristelajuarez1/cristelResume.git
```

2. Abre el archivo `index.html` en tu navegador

---

## 📦 Deploy

Este proyecto está desplegado utilizando **GitHub Pages**.

Para hacer deploy:

```bash
git add .
git commit -m "update"
git push origin main
```

Luego GitHub Pages publica automáticamente el sitio.

---

## ✨ Personalización

Puedes modificar fácilmente:

* Estilos en `assets/css/`
* Imágenes en `assets/images/`
